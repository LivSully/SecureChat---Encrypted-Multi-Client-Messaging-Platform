Welcome to Cool Secure Chat! (CSC for short)
Change the ip to match the server's machine, and run the server code
Then, several friends can run the UserLoginGUI file in order to sign in and join the server!
In order to sign in, username and password must exist in the credentials.txt file in the format username,password
The gui will display our logo, several interactive features such as typing fields, chatroom options, login/send/clear buttons,
and a status indicator which shows your connection status to the server!
